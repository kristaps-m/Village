About Our Application

Welcome to our awesome full-stack application built by Kristaps M.

Feature 1: REST API

REST API using ASP.NET Core with three data types: House, Apartment, and Resident. And they have support CRUD operations.

Feature 2: Real-Life Scenario

  • Data processing logic is moved from controllers to services.
  • SQL database is connected.
  • Created default database entries, including houses, apartments, and residents.
  • Configured database relationships between models (e.g., one resident can have multiple apartments).
  • Added an "IsOwner" boolean field to the Resident model.
  • Created DTO models.
  • Integrated AutoMapper.

Feature 3: Client-Side Application

Created a client-side application using Angular and Bootstrap 5 with the following views:

  • All Houses: Display a table of all houses, and allow users to click on a house to view its details.
  • House Details (id=1): Show details of a house, including its apartments. Users can edit the house details and view a list of residents. Clicking on a resident opens a modal for editing.
  • Apartment Details (id=1): Display apartment details, including residents. Users can edit the apartment details. Clicking on a resident opens a modal for editing.

Feature 4: Authentication

Implemented OAuth 2.0 authentication to allow users to log in with different roles: Manager and Resident. Different information will be visible based on the role.
